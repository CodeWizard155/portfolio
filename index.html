SpamShield

SpamShield is a web-based spam detection tool that identifies unwanted SMS and email messages using machine learning. Users can log in securely and choose between models like Naive Bayes (Accuracy: 95.55%), Random Forest (Accuracy: 97.49%), and Voting Classifier (Accuracy: 94.97%) for SMS classification. The app features a clean Streamlit interface, real-time predictions, and is designed for future expansion to include call spam detection.
